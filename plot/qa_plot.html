ABIDE: Anatomical Data Quality Metrics

Here we plot all the anatomical QA measures. The code for this can be found at https://github.com/preprocessed-connectomes-project/abide

Load Dependencies

Functions and libraries that are needed for plotting needs

source("qa_plot_funcs.R")

Read in Data

Along with reading the data, we setup descriptions that will be associated with each column and used as the label for the y-axis.

df <- read.csv("Phenotypic_V1_0b_preprocessed.csv")
qa.measures <- colnames(df)[grep("^anat_", colnames(df))]
qa.descs <- list(anat_cnr = "Contrast to Noise Ratio (CNR)", anat_efc = "Entropy Focus Criteria (EFC)", 
    anat_fber = "Foreground to Background Energy Ratio (FBER)", anat_fwhm = "FWHM in mm", 
    anat_qi1 = "Percent of Artifact Voxels (Qi1)", anat_snr = "Signal to Noise Ratio (SNR)")

Plot each measure

Now we plot the data. Note that here we are removing outliers with values greater than 3 times the IQR relative to the 25% or 75% mark.

for (measure in qa.measures) {
    desc <- qa.descs[[measure]]
    plot_measure(df, measure, desc, site.col = "SITE_ID", plot = TRUE, outfile = NULL, 
        rm.outlier = TRUE)
}
